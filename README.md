# 🩺 Medikart – Online Medical Platform (Frontend)

**Medikart** is a responsive frontend web application built to represent an online medical store.  
It allows users to explore healthcare products and medicines through a clean and user-friendly interface.

---

## 🌟 Features

- 💊 **Modern UI Design** – Clean, simple, and user-focused layout  
- 📱 **Fully Responsive** – Works smoothly on mobile, tablet, and desktop  
- 🔍 **Search & Browse** – Easy navigation for products and categories  
- 🧭 **Smooth Navigation** – Interactive menus and transitions  
- 💬 **Static Chatbot UI (Optional)** – Placeholder for future AI assistant integration  

---

## 🧩 Tech Stack

| Purpose | Technology |
|:--------|:------------|
| **Frontend** | HTML5, CSS3, JavaScript |
| **UI Design** | Custom CSS & Flexbox/Grid |
| **Version Control** | Git & GitHub |
| **Deployment (optional)** | GitHub Pages / Netlify |
