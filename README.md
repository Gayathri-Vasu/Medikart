# 🩺 Medikart – Online Pharmacy Management System

Medikart is a modern web-based pharmacy management application designed to simplify the process of purchasing medicines online. The platform allows users to browse and order both traditional and modern medicines through an intuitive and responsive interface while providing an efficient management system for pharmacy operations.

---

## 🚀 Project Overview

The primary goal of Medikart is to digitize pharmacy services by providing a user-friendly platform where customers can easily explore medicines, manage prescriptions, and place orders online. The system aims to improve accessibility, reduce manual effort, and enhance the overall healthcare shopping experience.
